How to read this. On books now = covers currently held for that service month. LY same point = covers held for the equivalent month at the same lead time last year. Pace = now vs LY same point (green ahead, red behind). No-show = trailing settled no-show rate for that shift/channel; OTA channels (OpenTable, Google Reserve) run far higher than the website booking widget. The Reliability tab splits on-books into expected cancellations, expected no-shows and net expected covers. Forecast = LY final + (now − LY same point), a conservative guide — months still fill materially in-month, so early months read low. Dim Sum Library opened ~Dec 2025 so its year-on-year is not like-for-like: it is shown on-books only, kept out of the group total, and its pace/forecast marked n/a.
