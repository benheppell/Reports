This page is a static snapshot of the numbers as of 1 Jul 2026. To pull fresh bookings, ask for the tracker to be regenerated (it re-queries SevenRooms and rebuilds the page).
How to read this. On books now = covers currently held for that service month. LY same point = covers held for the equivalent month at the same lead time last year. Pace = now vs LY same point (green ahead, red behind). Forecast = LY final + (now − LY same point), a conservative guide — months still fill materially in-month, so early months read low. Aqua Kyoto NYC opened in 2025, so its year-on-year is not like-for-like: it is shown on-books only, kept out of the group total, and its pace/forecast marked n/a.
GA4-independent · source: SevenRooms reservations (Supabase). Covers = reservations.covers. Shift split by seating time; booking-channel view excludes walk-ins.
